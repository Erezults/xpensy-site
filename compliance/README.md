# Compliance — חוקי אתר/אפליקציה עסקיים בישראל

מסמך צ׳קליסט: **3 חוקים שכל אתר עסקי בישראל חייב לעמוד בהם** (Brain by Eden Bibas).

1. **נגישות** — תקן ישראלי 5568 (רמה AA): הצהרת נגישות, רכיב נגישות עם פרטי קשר, טקסט חלופי (alt), ניגודיות 4.5:1, ניווט מלא במקלדת.
2. **דיוור** — חוק הספאם (תיקון 40): הסכמה נפרדת (צ׳קבוקס לא מסומן מראש), תיעוד הסכמה, הסרה בכל דיוור, זיהוי שולח, סימון "פרסומת".
3. **פרטיות** — חוק הגנת הפרטיות (תיקון 13, אוגוסט 2025): מדיניות פרטיות, אישור לפני שליחה, הודעת עוגיות, פירוט המידע הנאסף, אבטחת מידע.
